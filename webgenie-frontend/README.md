
  # WebGenieUI


  This is the Client-Side Code Bundle for the WebGenie Web-based GRN Platform built upon the Beeline Benhmarking Algorithm. It include features such as:
  
  * Dataset's Metadata Navigation
  * Implementation of the contiuous "size range" in the dashboard page
  * Provision for integration of the fastAPI backend though web socket.
  * Changes are Dynamically Loaded from a postgreSQL database
  * Provision made for cloud deployment to Azure.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
